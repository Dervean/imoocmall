vue v.s. react

轻量
响应式组件
服务端渲染
易于集成路由工具、打包工具和状态管理工具

vue
    模版、渲染函数的弹性选择
    简单的语法与项目创建
    渲染速度更快、体积更小

react
    大型企业应用、可测试性
    同时使用 web 和 app
    更多的支持和工具
    
vue
    model 绑定 view
    没有控制器的概念
    数据驱动、状态管理
